-Получение списка клиентов, в т.ч. по оказываемым услугам в заданном интервале времени, задействованным служащим и пр.
<index> -> Список клиентов -> <clients>( -> Фильтр -> <client_filter> -> Сохранить/отмена -> <clients>)

-Получение списка служащих по их участию в оказании услуг заданным клиентам и в заданное время
<index> -> Список сотрудников -> <employees>( -> Фильтр -> <employee_filter> -> Сохранить/отмена -> <employees>)

-Регистрация договора об оказании услуги
<index> -> Список услуг -> <services> -> Добавить услугу -> <service_modify> -> Сохранить -> <services>

-Добавление и удаление данных о клиенте, чтение и редактирование данных о нем
<index> -> Список клиентов -> <clients> -> Список клиентов -> <client_show> -> (аналогичное добавление/удаление/модификация данных о контактных лицах и их телефонов/емаилов)
                                        -> Добавление/изменение данных о клиенте -> <client_modify> -> Сохранить/отмена -> <clients>
                                        -> Удаление клиента

-Добавление и удаление служащего, чтение и редактирование данных о нем
<index> -> Список сотрудников -> <employees> -> Список сотрудников -> <employee_show> -> (аналогичное добавление/удаление/модификация телефонов/емаилов)
                                             -> Добавление/изменение данных о клиенте -> <employee_modify> -> Сохранить/отмена -> <employees>
                                             -> Удаление клиента
